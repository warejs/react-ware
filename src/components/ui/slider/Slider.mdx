# UI: Slider

Slider control with a handle that can be moved up and down (vertical slider) or right and left (horizontal slider) on a bar to select a value (or a range if two handles are present)

```jsx
import * as RW from 'reactware';
```

<Playground className="Rw_UI_Slider dark_theme">
  <Rw.UI.Slider 
    max="100" 
    onChange={(value) => console.log(value)}
    step={10}
  />
</Playground>

<Playground className="Rw_UI_Slider dark_theme">
