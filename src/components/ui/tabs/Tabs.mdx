# UI: Tabs

Tabs is a graphical control element that allows multiple documents or panels to be contained within a single window.

```jsx
import * as RW from 'reactware';
```

<Playground className="Rw_UI_Tabs dark_theme" style={{background: '#3a3a3a'}}>
  <Rw.UI.Tabs
    active={0}
    items={[
      {label: 'Properties', content: <h1>Heading tab</h1>}, 
      {label: 'History', content: <p>Some regular paraghagh text</p>},
      {label: 'Clone Source', content: 'just test'},
      {label: 'Events', content: 'just test'},
    ]}
    onChange={(item, event) => console.log(item, event)}
  />
</Playground>

Alternitevely you can space out the tabs with `spaced` attribute.

<Playground className="Rw_UI_Tabs dark_theme" style={{background: '#3a3a3a'}}>

      {label: 'Clone Source', content: 'just test'},
      {label: 'Events', content: 'just test'},
