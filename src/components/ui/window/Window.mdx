# UI: Window

Window is a graphical control element consisting of a visual area containing some of the graphical user interface elements of the program it belongs to.

```jsx
import * as RW from 'reactware';
```

<Playground className="Rw_UI_Window dark_theme" style={{height: '400px', background: '#3a3a3a'}}>
  <Rw.UI.Window index={0} header="Window">
    <Rw.UI.Label>My Window</Rw.UI.Label>
  </Rw.UI.Window>
</Playground>

> While the above output a window, windows should be added to the [App](#app) container, where they can be managed. See example bellow.

<Playground style={{height: '400px'}}>
  <Rw.App windows={[
    { header: 'Settings', top: 25, left: 90},
    { header: 'Actions', left: 125, top: 55, },
    { header: 'Layers', minwidth: 50, top: 15, left: 15, width: 50,  },
  ]}>
  </Rw.App>
</Playground>
