# UI: Menus

Menu is a control with multiple actions which can be clicked upon to choose a selection to activate.

```jsx
import * as RW from 'reactware';
```

The `items` is array of objects that contains:

```
items: Array<{ 
  "label": "Any", 
  "shortcut": "Any", 
  "icon": "Any", 
  "items": "Array<Object>" // same as parent;
}>
```

## UI: Context Menu

Context Menu - a type of menu whose contents depend on the context or state in effect when the menu is invoked

<Playground className="Rw_UI_Menu_Context dark_theme" style={{height: '200px'}}>
  <Rw.UI.Menus.Context
    visable={true}
    top="10px"
    left="10px"
    items={[
      { label: 'Cut' },
      { label: 'Copy', items: [
        { label: 'All' }, 
        { label: 'Selected' }, 
        { label: 'First' }
      ]},
      { label: 'Selected', icon: "✓", shortcut: "CTRL+X" },
    ]}
  />
</Playground>

> While the above output a context menu, context item should be added to the [App](#app) container, where they can be referenced and autogenerated based on users right click. See example bellow.

<Playground style={{height: '200px'}}>
  <Rw.App menus={{
    'a-1': [
      { label: 'File' },
      { label: 'Start', items: [
        { label: 'Select' }, 
        { label: 'Maybe' }, 
        { label: 'Dont' }
      ]},
      { label: 'Exit' },
    ],
    'b-1': [
      { label: 'One' },
      { label: 'Two' },
      { label: 'Tree' },
      { label: 'Four' },
      { label: 'Five' },
    ],
  }}>
    <div>
      <button contextMenu="a-1">Context A</button>
      <button contextMenu="b-1">Context B</button>
      <button>No Context Menu</button>
    </div>
  </Rw.App>
</Playground>
