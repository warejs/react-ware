# UI: Menu

Context Menu - a type of menu whose contents depend on the context or state in effect when the menu is invoked

```jsx
import * as RW from 'reactware';
```

<Playground className="Rw_UI_Menu_Context dark_theme" style={{height: '200px'}}>

    visable={true}
    top="10px"
    left="10px"
    items={[
      { label: 'Cut' },
      { label: 'Copy', items: [
        { label: 'All' }, 
        { label: 'Selected' }, 
        { label: 'First' }
      ]},

While the above output a context menu, context item are added to the [App](#app) App container, where they can be referenced and autogenerated based on users right click. See example bellow.

<Playground style={{height: '200px'}}>
  <Rw.App contextMenus={{
    'a-1': [
      { label: 'File' },
      { label: 'Start', items: [
        { label: 'Select' }, 
        { label: 'Maybe' }, 
        { label: 'Dont' }
      ]},
      { label: 'Exit' },
    ],
    'b-1': [
      { label: 'One' },
      { label: 'Two' },
      { label: 'Tree' },
      { label: 'Four' },
      { label: 'Five' },
    ],
  }}>
    <div>
      <button contextMenu="a-1">Context A</button>
      <button contextMenu="b-1">Context B</button>
      <button>No Context Menu</button>
    </div>
  </Rw.App>
</Playground>
