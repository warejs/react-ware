## Cycle

Cycle button is a button that cycles its content through two or more values

<Playground className="Rw_UI_Buttons_Cycle dark_theme">
  <Rw.UI.Buttons.Cycle
    default={1} 
    options={[
      'state 1',
      'state 2',
      'state 3',
    ]}
    onCycle={(index, option, event) => console.log("selected", index, option)}
  />
</Playground>

<PropsTable of={Rw.UI.Buttons.Cycle} />

## Radio

## Split

Split Button control combining a button (typically invoking some default action).

<Playground className="Rw_UI_Buttons_Split dark_theme">
  <Rw.UI.Buttons.Split
    default={1} 
    options={[
      { label: 'Save', onClick: () => alert('saved') },
      { label: 'Publish', onClick: () => alert('published') },
      { label: 'Exit', onClick: () => alert('edit') }
    ]}
    onSelect={(index, option) => alert("selected", index, option)}
  />
</Playground>

<PropsTable of={Rw.UI.Buttons.Split} />
