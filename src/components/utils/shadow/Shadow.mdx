# Utils: Shadow

Shadow is a shadow creator util for React portals.

```jsx
import * as RW from 'reactware';
```

Example of component displaying in shadow, (does't contain styled).

    </Rw.Layouts.Float>
  </Rw.Utils.Shadow>
