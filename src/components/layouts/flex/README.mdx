## Flexible children with horizontal layout

Specific children can utalise more space (given priority) over siblings.

<Playground className="warelayoutflex">
  <WareLayoutFlex horizontal>
    <div>One</div>
    <WareLayoutFlex auto>Two</WareLayoutFlex>
    <div>Three</div>
  </WareLayoutFlex>
</Playground>

## Flexible ratios

Unlike bootstrap, the rations range between 1-12, but are only proprities and no scalable paters with dynamic resizing.

<Playground className="warelayoutflex">
  <WareLayoutFlex horizontal>
    <WareLayoutFlex ratio="1">1</WareLayoutFlex>
    <WareLayoutFlex ratio="2">2</WareLayoutFlex>
    <WareLayoutFlex ratio="3">3</WareLayoutFlex>
    <WareLayoutFlex ratio="6">6</WareLayoutFlex>
  </WareLayoutFlex>
</Playground>

## Self alignment

Child items have the ability to be slef aligned, regardless of the parent aplighment..

Self alignments: `start`, `center`, `end`,

<Playground className="warelayoutflex">
  <WareLayoutFlex horizontal style={{height:"150px"}}>
    <WareLayoutFlex self start>Start</WareLayoutFlex>
    <WareLayoutFlex self center>Center</WareLayoutFlex>
    <WareLayoutFlex self end>End</WareLayoutFlex>
    <WareLayoutFlex justified end>Justified end</WareLayoutFlex>
  </WareLayoutFlex>
</Playground>
