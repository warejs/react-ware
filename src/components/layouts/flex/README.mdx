## Flexible children with horizontal layout

Specific children can utalise more space (given priority) over siblings.

<Playground className="Rw.Layouts.Flex">
  <Rw.Layouts.Flex horizontal>
    <div>One</div>
    <Rw.Layouts.Flex auto>Two</Rw.Layouts.Flex>
    <div>Three</div>
  </Rw.Layouts.Flex>
</Playground>

## Flexible ratios

Unlike bootstrap, the rations range between 1-12, but are only proprities and no scalable paters with dynamic resizing.

<Playground className="Rw.Layouts.Flex">
  <Rw.Layouts.Flex horizontal>
    <Rw.Layouts.Flex ratio="1">1</Rw.Layouts.Flex>
    <Rw.Layouts.Flex ratio="2">2</Rw.Layouts.Flex>
    <Rw.Layouts.Flex ratio="3">3</Rw.Layouts.Flex>
    <Rw.Layouts.Flex ratio="6">6</Rw.Layouts.Flex>
  </Rw.Layouts.Flex>
</Playground>

## Self alignment

Child items have the ability to be slef aligned, regardless of the parent aplighment..

Self alignments: `start`, `center`, `end`,

    <Rw.Layouts.Flex self start>Start</Rw.Layouts.Flex>
    <Rw.Layouts.Flex self center>Center</Rw.Layouts.Flex>
    <Rw.Layouts.Flex self end>End</Rw.Layouts.Flex>
