# Reactware

Reactware is an open source toolkit for developing desktop PWA with HTML, CSS, and JS

## Install and Usage

```
npm i reactware --save
```

### Usage
```
import { WareLayoutFlex } from 'reactware';

<WareLayoutFlex horizontal>
  <div>One</div>
  <div>Two</div>
  <div>Three</div>
</WareLayoutFlex>
```
